
# Remede Agency

This repository contains all the source code to run the Argent Bank application.


## Run App

Clone the Frontend

```bash
  https://github.com/MalikMoulat/ArgentBank-Project-13.git
```

Go to the project directory

```bash
  cd front-end
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
